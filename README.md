# 🚀 Muhammad Zain-ul-Abdin — Portfolio

A modern, dark-themed personal portfolio website built with vanilla HTML, CSS & JavaScript. Features smooth animations, custom cursor, typing effects, and fully responsive design.

## ⚡ Live Demo

[View Live →](https://zaini12121.github.io/portfolio/)

## ✨ Features

- 🎨 **Dark Green Neon Theme** — Premium, modern aesthetic
- ✍️ **Typing Effect** — Dynamic role showcase in hero section
- 🖱️ **Custom Cursor** — Interactive cursor with hover effects
- 🌟 **Floating Particles** — Ambient particle animation
- 📱 **Fully Responsive** — Works on all devices (mobile, tablet, desktop)
- 🍔 **Mobile Menu** — Smooth hamburger navigation
- 📊 **Animated Stats** — Counter animation on scroll
- 📈 **Skill Bars** — Animated progress indicators
- ⏬ **Scroll Reveal** — Elements animate as you scroll
- 📋 **Contact Form** — Ready for FormSubmit integration
- 🎯 **Active Nav** — Navigation highlights current section
- ⬆️ **Scroll to Top** — Quick return button
- 🔍 **SEO Optimized** — Meta tags, semantic HTML

## 🚀 Deploy on GitHub Pages

1. Create a new repository on GitHub
2. Push this code to the repository:
   ```bash
   git init
   git add .
   git commit -m "Initial commit - Portfolio Website"
   git branch -M main
   git remote add origin https://github.com/yourusername/portfolio.git
   git push -u origin main
   ```
3. Go to **Settings** → **Pages**
4. Under "Source", select **main** branch
5. Click **Save** — your site will be live at `https://yourusername.github.io/portfolio/`

## ⚙️ Customize

### Update Social Links
In `index.html`, find the contact section and update these links:
- Email: `mailto:your@email.com`
- LinkedIn: `https://linkedin.com/in/your-profile`
- GitHub: `https://github.com/your-username`
- Instagram: `https://instagram.com/your-handle`

### Contact Form
The form uses [FormSubmit](https://formsubmit.co/). To activate:
1. Replace `your@email.com` in the form action URL
2. Remove the `preventDefault()` in `main.js` (line ~230)
3. Submit the form once to activate your email

### Add Resume
Replace `href="#"` on the Resume button with a link to your PDF file.
